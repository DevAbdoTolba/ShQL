# ShQL
Bash Script-Based Database Management System

## Description
Database and Management system made by bashscript to understand the methodolgy of folder/file hierarchy of linux.
A Directory is a database represntation, and each file is considred as a table in that database.

The project is using Makefile to setup/install/clean/help.
